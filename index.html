WALK-IN EXAM SYSTEM

A Java-based web application was developed for Sukhothai Thammathirat Open University. This project included several sub-module web applications, such as Registration, Exam Form Creator, Exam Result Analysis, and Electronic Examination. The web application was built on the JSF framework and used Hibernate to connect to MSSQL. Additionally, there was an integration module (web service) that facilitated communication between the WALK-IN EXAM System and the Examination Register System using REST API.
